Phonograph Audio Visualizer

Description: The Phonograph Audio Visualizer project was completed as a project for a Web Development course. It utilizes the HTML canvas to display the phonograph and the audio itself. Image processing using this system allowed image effects such as tint or noise to be possible as well.
The audio is processed using HTML 5's Web Audio API. This allowed me to create multiple audio effect nodes that change the audio before the user hears it, such as a shelf that amplifies certain frequencies and gain to raise the volume.
The GUI was created mostly using the structure provided by the dat.gui framework. However, I did end up using JavaScript to change a dropdown list to a radio-button group to improve readability.

Role: Sole Designer - I designed this project completely on my own using resources from the class and through my own research.

Language(s) Used: JavaScript, HTML, CSS

System(s) Used: HTML Canvas, Web Audio API, Dat.GUI
