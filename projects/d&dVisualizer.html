Dungeons & Dragons Class Visualizer

Description: The Dungeons & Dragons Clas Visualizer was created as a project for a Data Visualization course. It utilizes d3.js to read, analyze, and organize the data, as well as displaying the data itself.
Each class is mapped to a color and that color is displayed in a circle. The circles themselves are also bound by forces so that they seem somewhat organic. When a class or circle is clicked on, the visualization zooms in to display the chosen race as well. A popup also appears while hovering over a circle that reveals the statistics of that player.

Role: Sole Developer - I designed this project completely on my own using resources from the class and through my own research.

Language(s) Used: JavaScript, HTML, CSS

System(s) Used: d3.js
